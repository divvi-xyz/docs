## Adding Moderators

Channel owners can assign moderators to help maintain content quality, enforce guidelines, and manage user interactions:

```typescript
const addModeratorResult = await sageClient.addModerator({
  amounts: [0, 0],                             // Required protocol fees
  appId: 'your-app-id',                        // Explicitly required application ID
  network: 'testnet',                          // 'testnet' or 'mainnet'
  channelId: 'channel-id',                     // Channel to add moderator to
  sharedUserId: 'moderator-user-shared-id',    // User to grant moderator role
  self: walletAddress                          // Channel owner's wallet address
});

if (addModeratorResult.ok) {
  await signAndExecuteTransaction({ transaction: addModeratorResult.transaction });
} else {
  console.error(addModeratorResult.err);
}
```

## Removing Moderators

Channel owners can also remove moderators:

```typescript
const removeModeratorResult = await sageClient.removeModerator({
  amounts: [0, 0],                             // Required protocol fees
  appId: 'your-app-id',                        // Explicitly required application ID
  network: 'testnet',                          // 'testnet' or 'mainnet'
  channelId: 'channel-id',                     // Channel to remove moderator from
  sharedUserId: 'moderator-user-shared-id',    // User whose moderator role will be revoked
  self: walletAddress                          // Channel owner's wallet address
});

if (removeModeratorResult.ok) {
  await signAndExecuteTransaction({ transaction: removeModeratorResult.transaction });
} else {
  console.error(removeModeratorResult.err);
}
```
