---
title: "Unfollowing Channels"
description: " "
---

Users can leave channels anytime to stop receiving updates or participating:

```typescript
const unfollowChannelResult = await sageClient.unfollowChannel({
  amounts: [0, 0],                      // Required protocol fee (default: [0,0])
  channelId: 'channel-id',              // Channel to unfollow
  self: walletAddress                   // User's wallet address
});

if (unfollowChannelResult.ok) {
  await signAndExecuteTransaction({ transaction: unfollowChannelResult.transaction });
} else {
  console.error(unfollowChannelResult.err);
}
```
