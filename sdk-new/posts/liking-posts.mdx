Likes explicitly express appreciation and engagement. Likes improve content visibility and trigger incremental rewards.

```typescript
const likeResult = await sageClient.likePost({
  amounts: [0, 0],                               // Required protocol fees
  appId: 'your-app-id',                          // Explicitly required application ID
  network: 'testnet',                            // 'testnet' or 'mainnet'
  authorSharedUserId: 'original-author-shared-id', // Post author's shared user ID
  postId: 'target-post-id',                      // Post to like
  ownedUserId: 'your-owned-user-id',             // Liker’s owned user ID
  self: walletAddress                            // Liker’s wallet address
});

if (likeResult.ok) {
  await signAndExecuteTransaction({ transaction: likeResult.transaction });
} else {
  console.error(likeResult.err);
}
```
