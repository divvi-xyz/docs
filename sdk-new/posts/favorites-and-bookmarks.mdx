title: "Favorites and Bookmarks"
description: " "
---

Users can bookmark posts for easy reference:

**Adding a Favorite Post**:

```typescript
await sageClient.addFavoritePost({
  appId: 'your-app-id',                          // Required application ID
  network: 'testnet',                            // 'testnet' or 'mainnet'
  authorSharedUserId: 'author-shared-user-id',   // Author's shared user ID
  postId: 'favorite-post-id',                    // Post to favorite
  ownedUserId: 'your-owned-user-id'              // Your owned user ID
});
```

**Removing a Favorited Post**:

To remove bookmarks, use the corresponding removal method:

```typescript
await sageClient.removeFavoritePost({
  appId: 'your-app-id',                          // Required application ID
  network: 'testnet',                            // 'testnet' or 'mainnet'
  postId: 'favorite-post-id',                    // Post to remove from favorites
  ownedUserId: 'your-owned-user-id'              // Your owned user ID
});
```
