Comments in Sage Protocol enable clear, structured conversations around blockchain-native posts. Using Sage SDK’s React hooks, developers can effortlessly implement threaded discussions, making interactions intuitive and explicitly traceable within decentralized communities.

## Creating Comments (React)

Comments are blockchain-native posts explicitly linked to parent posts. Here's how to create a comment:

```typescript
import { usePost } from '@sageprotocol/sdk/react';
import { useWallet } from '@suiet/wallet-kit';

function CreateComment({ parentPostId, authorSharedUserId, ownedUserId }) {
  const { commentOnPost } = usePost();
  const { account, signAndExecuteTransaction } = useWallet();

  async function handleCreateComment() {
    const result = await commentOnPost({
      amounts: [0, 0],
      appId: 'your-app-id',
      network: 'testnet',
      authorSharedUserId,           // Original post author's shared user ID
      parentPostId,                 // ID of the parent post
      title: 'Great perspective!',
      description: 'My thoughts on this...',
      data: 'I completely agree, particularly about scalability...',
      ownedUserId,
      self: account.address
    });

    if (result.ok) {
      await signAndExecuteTransaction({ transaction: result.transaction });
      console.log("Comment created successfully!");
    } else {
      console.error(result.err);
    }
  }

  return <button onClick={handleCreateComment}>Comment</button>;
}
```

## Displaying Threaded Comments (React)

Comments form explicit discussion threads. Clearly structuring these threads is key for readability. Here's an explicit pattern for rendering threaded comments:

```typescript
type Comment = {
  id: string;
  author: string;
  title: string;
  description: string;
  data: string;
  replies?: Comment[];
};

function CommentThread({ comments }: { comments: Comment[] }) {
  return (
    <div>
      {comments.map((comment) => (
        <div key={comment.id} style={{ marginLeft: 20 }}>
          <strong>{comment.title}</strong> by {comment.author}
          <p>{comment.data}</p>
          {comment.replies && <CommentThread comments={comment.replies} />}
        </div>
      ))}
    </div>
  );
}
```

Example Data Structure:

```typescript
const comments = [
  {
    id: '1',
    author: 'Alice',
    title: 'Great insights!',
    description: 'Agree on scalability',
    data: 'Your points on scalability were insightful...',
    replies: [
      {
        id: '2',
        author: 'Bob',
        title: 'Further thoughts',
        description: 'Adding to scalability',
        data: 'I’d like to add more details...',
      },
    ],
  },
];
```
