---
title: "Managing Favorites"
description: " "
---

Users can favorite preferred channels, posts, and other users to maintain personalized collections.

### Favorite Channels

```typescript
await sageClient.addFavoriteChannel({
  appId: 'your-app-id',                       // required
  channelId: 'target-channel-id',             // required
  network: 'testnet',                         // 'testnet' or 'mainnet', required
  ownedUserId: 'your-owned-user-id'           // required
});
```

### Favorite Posts

```typescript
await sageClient.addFavoritePost({
  appId: 'your-app-id',                       // required
  authorSharedUserId: 'author-shared-user-id',// required
  network: 'testnet',                         // 'testnet' or 'mainnet', required
  ownedUserId: 'your-owned-user-id',          // required
  postId: 'target-post-id'                    // required
});
```

### Favorite Users

```typescript
await sageClient.addFavoriteUser({
  appId: 'your-app-id',                       // required
  favoriteSharedUserId: 'favorite-user-shared-id', // required
  network: 'testnet',                         // 'testnet' or 'mainnet', required
  ownedUserId: 'your-owned-user-id'           // required
});
```

**Removing favorites follows a similar pattern** using corresponding remove methods:

- `removeFavoriteChannel`
- `removeFavoritePost`
- `removeFavoriteUser`
