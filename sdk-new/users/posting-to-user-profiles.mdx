Users can directly post content onto other users' profiles, ideal for endorsements, community appreciation, or personal interactions:

```typescript
const postResult = await sageClient.postToUser({
  amounts: [0, 0],                               // Required protocol fee amounts, usually [0,0]
  data: 'Your contributions to the project were outstanding!', // Main markdown content
  description: 'Endorsement for exceptional work',             // Summary in markdown
  ownedUserId: 'your-owned-user-id',             // Posterâ€™s owned user object
  self: walletAddress,                           // Wallet address of the poster
  sharedUserId: 'target-user-shared-id',         // Target user's shared user object
  title: 'ðŸŒŸ Professional Recommendation'        // Post headline/title in markdown
});

if (postResult.ok) {
  await signAndExecuteTransaction({ transaction: postResult.transaction });
} else {
  console.error(postResult.err);
}
```
