# Documentação de Integração da Prefeitura do Rio

Esta documentação descreve a infraestrutura de integração desenvolvida pela IplanRio para a Prefeitura do Rio de Janeiro, conectando os diversos sistemas e secretarias municipais.

## Visão Geral

A **Infraestrutura de Integração da Prefeitura do Rio** é um ecossistema completo que conecta, processa e disponibiliza dados e serviços entre as diferentes secretarias municipais. Nossa plataforma oferece uma arquitetura moderna, segura e escalável para integração de sistemas, processamento de dados e disponibilização de APIs.

Esta documentação serve como guia central para desenvolvedores, arquitetos e gestores que trabalham com a infraestrutura de integração da Prefeitura.
